# PWA 图标设置指南

## 步骤说明

1. **下载提供的图标图片**
   - 保存为PNG格式

2. **创建两个尺寸的图标**
   - 192x192像素：`icon-192x192.png`
   - 512x512像素：`icon-512x512.png`

3. **推荐工具**
   - 在线工具：
     - [ResizeImage.net](https://resizeimage.net/)
     - [PicResize](https://picresize.com/)
   - 桌面工具：
     - Photoshop
     - GIMP (免费)
     - Paint.NET (免费)

4. **放置位置**
   - 将生成的两个图标文件放在此文件夹中

5. **验证**
   - 确保文件名正确：
     - `icon-192x192.png`
     - `icon-512x512.png`
   - 确保文件格式为PNG

## 注意事项

- 保持图标清晰，避免模糊
- 确保图标在不同尺寸下都能正常显示
- 图标背景建议使用透明或与应用主题色一致的颜色
