# Rock的减脂小工具 - 产品需求文档

## 概述
- **摘要**：Rock的减脂小工具是一款科学的减脂计算器应用，帮助用户根据个人身体数据和减脂目标，计算每日热量摄入建议、减脂计划和相关健康指标。
- **目的**：解决用户在减脂过程中缺乏科学指导的问题，提供个性化的减脂方案和数据支持。
- **目标用户**：希望通过科学方法进行减脂的健康人群，包括健身爱好者、体重管理人群和健康生活方式追求者。

## 目标
- 提供准确的身体数据计算和减脂计划生成
- 设计直观、美观的用户界面，提升用户体验
- 支持离线使用，确保用户随时随地访问工具
- 提供详细的体脂率参考和视觉指导
- 个性化的减脂建议和点评

## 非目标（范围外）
- 不提供饮食计划和具体食谱
- 不提供运动计划和训练指导
- 不连接外部健康设备或API
- 不存储用户数据到云端
- 不提供社交分享功能

## 背景与上下文
- **技术背景**：使用React + TypeScript + Vite + Tailwind CSS构建，支持PWA功能
- **市场背景**：越来越多的人关注健康和体重管理，但缺乏科学的减脂指导工具
- **用户需求**：用户需要简单易用、科学准确的减脂计算工具，帮助制定合理的减脂计划

## 功能需求
- **FR-1**：身体基础数据输入，包括性别、年龄、身高、体重和日常活动量
- **FR-2**：减脂目标设定，包括当前体脂、目标体脂和计划用时
- **FR-3**：科学计算功能，包括基础代谢（BMR）、每日总消耗（TDEE）、BMI、目标体重、需减脂肪量等
- **FR-4**：减脂计划生成，提供不同强度的减脂方案选择
- **FR-5**：体脂率视觉参考，支持点击查看大图
- **FR-6**：脂肪减量卡片，显示详细的脂肪减量信息
- **FR-7**：整体点评功能，提供个性化的减脂建议和教练寄语
- **FR-8**：PWA支持，包括离线使用和本地安装

## 非功能需求
- **NFR-1**：响应式设计，适配移动端和桌面端
- **NFR-2**：实时计算和动画效果，提升用户体验
- **NFR-3**：输入验证和格式化，确保数据准确性
- **NFR-4**：性能优化，确保应用加载和计算速度快
- **NFR-5**：隐私保护，所有数据仅存储在本地

## 约束
- **技术**：使用React + TypeScript + Vite + Tailwind CSS
- **平台**：支持现代浏览器，包括Chrome、Firefox、Safari、Edge
- **设备**：支持桌面端和移动端
- **数据**：所有数据仅存储在本地，不进行云端存储

## 假设
- 用户具备基本的健康知识和减脂常识
- 用户能够准确输入个人身体数据
- 用户理解减脂是一个长期过程，需要坚持和合理规划

## 验收标准

### AC-1：身体基础数据输入
- **给定**：用户打开应用
- **当**：用户输入性别、年龄、身高、体重和日常活动量
- **则**：应用实时验证输入数据，确保数据格式正确，并显示计算结果
- **验证**：`human-judgment`

### AC-2：减脂目标设定
- **给定**：用户已输入身体基础数据
- **当**：用户输入当前体脂、目标体脂和计划用时
- **则**：应用计算目标体重、需减脂肪量等数据，并更新计算结果
- **验证**：`human-judgment`

### AC-3：科学计算功能
- **给定**：用户输入了完整的身体数据和减脂目标
- **当**：应用进行计算
- **则**：应用显示准确的BMR、TDEE、BMI、目标体重、需减脂肪量等数据
- **验证**：`programmatic`

### AC-4：减脂计划生成
- **给定**：用户输入了完整的身体数据和减脂目标
- **当**：用户选择不同强度的减脂计划
- **则**：应用自动调整计划用时，并显示相应的减脂建议
- **验证**：`human-judgment`

### AC-5：体脂率视觉参考
- **给定**：用户打开应用
- **当**：用户点击体脂率参考图片
- **则**：应用显示大图模态框，用户可以查看详细的体脂率参考
- **验证**：`human-judgment`

### AC-6：脂肪减量卡片
- **给定**：用户输入了完整的身体数据和减脂目标
- **当**：应用计算完成
- **则**：应用显示脂肪减量卡片，包括需减脂肪量、相当于多少卡路里、汉堡数量和慢跑时间
- **验证**：`human-judgment`

### AC-7：整体点评功能
- **给定**：用户输入了完整的身体数据和减脂目标
- **当**：应用计算完成
- **则**：应用显示整体点评，包括计划强度、影响与意义、注意事项和教练寄语
- **验证**：`human-judgment`

### AC-8：PWA支持
- **给定**：用户使用支持PWA的浏览器
- **当**：用户访问应用
- **则**：应用提示用户安装，支持离线使用
- **验证**：`human-judgment`
