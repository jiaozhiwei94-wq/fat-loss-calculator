# 计划页优化实施计划

## 1. 仓库研究结论

当前计划页结构：
- 包含饮食计划和训练计划两个主要部分
- 饮食计划显示三大营养素的分配（蛋白质、碳水化合物、脂肪）
- 训练计划显示训练类型、建议时长、强度和预计运动消耗
- 页面底部有返回结果页按钮

## 2. 需要修改的文件

- `src/App.tsx` - 主要修改计划页的内容和布局

## 3. 优化步骤

### 3.1 饮食计划优化

1. **增加每日总热量目标**
   - 在饮食计划部分顶部添加每日热量摄入建议
   - 显示清晰的总热量数值，如："每日建议总热量摄入：1500 大卡"

2. **增加餐盘示例**
   - 添加文字描述，例如："早餐：2个鸡蛋 + 一小碗燕麦粥 + 少量蔬菜；午餐：一份鸡胸肉沙拉（100g 鸡胸肉）+ 一碗糙米饭；晚餐：鱼肉（150g）+ 大量蔬菜。"

3. **提供常见食物的宏量参考**
   - 添加食物宏量参考表，例如："100g 鸡胸肉 ≈ 25g 蛋白质；1个鸡蛋 ≈ 6g 蛋白质；1碗米饭 ≈ 40g 碳水。"

4. **增加注意事项**
   - 添加饮食注意事项，例如："请优先选择优质蛋白（鸡胸肉、鱼肉、鸡蛋），复合碳水（糙米、全麦面包），健康脂肪（牛油果、坚果）。"

### 3.2 训练计划优化

1. **训练类型具体化**
   - 有氧运动：改为"跑步、跳绳、快走、椭圆机等"
   - 力量训练：改为"深蹲、硬拉、卧推、划船、俯卧撑等（可在家或健身房进行）"

2. **增加周训练建议**
   - 添加小白周计划，例如："周一：力量训练（全身）；周二：休息；周三：有氧运动（跑步 45 分钟）；周四：力量训练（下肢）；周五：休息；周六：有氧运动（快走 60 分钟）；周日：休息。"

3. **增加强度建议**
   - 添加可操作性描述：例如，"中等强度：运动时能说话但不能唱歌，略微出汗。"

4. **增加注意事项**
   - 添加训练注意事项，例如："初学者请从轻重量开始，保证动作规范；每次运动前请热身 10 分钟，结束后拉伸 10 分钟。"

### 3.3 整体 UI/UX 优化

1. **增加达成天数**
   - 在页面顶部显示达成目标的预计天数

2. **卡片设计优化**
   - 为每个计划卡片添加"查看更多详情"或"获取个性化食谱/训练计划"按钮

3. **增加下一步引导**
   - 添加"下一步：开始你的减脂旅程"或"保存你的计划"按钮

4. **增加免责声明**
   - 在页面底部添加"本计算器提供科学参考，具体计划请咨询专业医生或教练"的声明

## 4. 潜在依赖和考虑因素

- 确保修改后的布局在移动端和桌面端都能正常显示
- 保持与现有设计风格的一致性
- 确保所有新增内容都能正确显示，不会影响页面加载速度

## 5. 风险处理

- 如果用户数据不完整，确保相关计算和显示能够优雅处理
- 确保新增内容不会导致页面布局混乱
- 确保所有链接和按钮功能正常

## 6. 实施顺序

1. 先修改饮食计划部分
2. 然后修改训练计划部分
3. 最后进行整体 UI/UX 优化
4. 测试所有功能是否正常工作
5. 确保在不同设备上的显示效果
